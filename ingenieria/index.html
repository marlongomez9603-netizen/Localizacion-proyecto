📐 Método SLP (Systematic Layout Planning)

El método SLP de Muther clasifica las relaciones entre áreas con valores de cercanía (A = absolutamente necesaria, E = especialmente importante, I = importante, O = ordinaria, U = no necesaria, X = indeseable). Esta cuadrícula es una representación simplificada; el diseño real considera flujos de materiales, seguridad y normativa vigente.

📉 Depreciación Línea Recta

Fórmula: Dep. anual = (Costo – Valor de rescate) / Vida útil. Aquí se asume valor de rescate = 0 para simplificar el cálculo. En el estudio financiero real se incorpora como gasto no desembolsable.

📋 Marco legal laboral

En Colombia, los contratos de trabajo se rigen por el Código Sustantivo del Trabajo (CST). Los cargos pueden vincularse mediante contrato a término fijo, indefinido, obra o labor, o prestación de servicios. El tipo de vinculación afecta la estructura de costos laborales (prestaciones sociales, aportes parafiscales, seguridad social).
